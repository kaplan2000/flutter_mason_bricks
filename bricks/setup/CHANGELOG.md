# Changelog

All notable changes to the `setup` brick will be documented in this file.

## [0.0.1] - 2025-12-28

### Added

- Comprehensive project structure in `__brick__/lib`.
- Automatic project name detection in `pre_gen` hook.
- Automatic dependency injection and configuration in `post_gen` hook.
- Automatic `flutter pub get` and `build_runner` execution.
- Standardized `README.md` and `widget_test.dart` generation.

### Fixed

- Hook compilation errors and directory structure.
